Данный репозиторий содержит комплекс аналитических моделей и решений прикладных задач, разработанных в рамках обучения по специальности «Автоматизированные системы обработки информации» (БГУИР). Все проекты реализованы в MS Excel и представляют собой моделирование реальных бизнес-ситуаций с использованием сложного математического аппарата.

## Часть 1. Методы принятия решений и системный анализ
**Файл:** `Decision_Making_Methods_Analysis.xlsx`

В этом блоке представлены модели для работы в условиях неопределенности, экспертной оценки и статистического прогнозирования.

### 1. Экспертный анализ в неструктурированных задачах
*   **Бизнес-задача:** Выбор оптимального способа привлечения капитала (кредит, аренда, продажа помещений или акций).
*   **Метод:** Агрегирование мнений группы экспертов, построение коллективной ранжировки.
*   **Результат:** Реализована модель учета субъективных оценок для выбора стратегии с наивысшим приоритетом.

### 2. Метод анализа иерархий (МАИ) и ExpertChoice
*   **Бизнес-задача:** Выбор надежного дистрибьютора из 6 кандидатов на основе репутации, опыта и торговой сети.
*   **Метод:** Метод анализа иерархий Томаса Саати.
*   **Результат:** Построены матрицы попарных сравнений, рассчитаны векторы приоритетов и проверена согласованность матриц.

### 3. Принятие решений при многих критериях в условиях риска
*   **Бизнес-задача:** Выбор технологической линии (Л1, Л2, Л3) для производства пластмасс.
*   **Метод:** Многокритериальная модель, учитывающая вероятностное распределение заказов.
*   **Результат:** Оптимизация выбора по критериям производительности, себестоимости и надежности.

### 4. Статистические методы и план закупок
*   **Бизнес-задача:** Формирование плана закупок 4-х видов товаров для максимизации доходности при минимальном риске.
*   **Результат:** Реализована модель поиска оптимального портфеля закупок с целевой доходностью 1.1 на единицу вложений.

### 5. Эконометрическое моделирование и прогнозирование
*   **Методы:** Парная и множественная линейная регрессия, корреляционный анализ.
*   **Результат:** Построена и проверена на адекватность (F-тест, t-тест) модель сбыта; выполнен точечный и интервальный прогноз с надежностью 95%.

### 6. Кластерный анализ (Сегментация объектов)
*   **Бизнес-задача:** Классификация стран по уровню жизни (доход и образование).
*   **Методы:** Нормировка данных, метод K-средних (K-means), метод максимина.
*   **Результат:** Автоматическая группировка объектов в кластеры по сходству признаков.

### 7. Критерии теории игр (Риск и неопределенность)
*   **Бизнес-задача:** Оптимизация производственного плана при неопределенном спросе.
*   **Критерии:** Байеса, Лапласа, Вальда (максимин), Сэвиджа, Гурвица.
*   **Результат:** Сравнительный анализ стратегий при различных психологических установках ЛПР.

## Часть 2. Оптимизационные методы и исследование операций
**Файл:** `Optimization_Methods_Operations_Research.xlsx`

В этом блоке представлены решения задач оптимизации ресурсов и процессов с использованием математического программирования.

### 1. Линейное программирование и симплекс-метод
*   **Бизнес-задача:** Оптимизация производственного плана при ограничениях по времени работы оборудования.
*   **Методы:** Симплекс-метод, двухэтапный метод (искусственного базиса), анализ чувствительности ресурсов.

### 2. Целочисленное программирование
*   **Бизнес-задача:** Расчет выпуска изделий с требованием целочисленности продукции.
*   **Метод:** Метод ветвей и границ (Branch and Bound).
*   **Особенность:** Алгоритм реализован вручную через последовательное добавление ограничений.

### 3. Транспортные задачи и логистика
*   **Бизнес-задача:** Минимизация затрат на перевозку угля от 3-х шахт к 4-м пунктам.
*   **Методы:** Метод минимального элемента, метод потенциалов.

### 4. Нелинейное программирование
*   **Бизнес-задача:** Минимизация экологического загрязнения при производстве хим. реактивов.
*   **Метод:** Метод Франка-Вульфа (итерационное нахождение минимума функции).

### 5. Динамическое программирование
*   **Бизнес-задача:** Оптимальное распределение инвестиций между четырьмя предприятиями.
*   **Метод:** Принцип оптимальности Беллмана.

### 6. Теория массового обслуживания (СМО)
*   **Бизнес-задача:** Анализ эффективности производственного участка как многоканальной системы.
*   **Результат:** Расчет интенсивности потоков, вероятностей простоя и прибыли участка за смену.

### 7. Задачи о назначениях (Распределение мощностей)
*   **Бизнес-задача:** Минимизация стоимости выполнения работ исполнителями.
*   **Метод:** Алгоритм перераспределения работ (Венгерский метод / Метод Мака).
